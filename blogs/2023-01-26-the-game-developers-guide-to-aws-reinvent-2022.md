<p>AWS re:Invent was back in full force in 2022 with over 50,000 members of the global cloud community coming together in Las Vegas for the 11th edition of the conference. With dozens of new products launched and hundreds of breakout sessions to attend, there is always too much for one person to cover in four days. This blog details the service launches, customer sessions, and partner announcements most relevant to game developers.</p> 
<h2>Service Launches</h2> 
<p>Across more than 100 service launches from Amazon Web Services (AWS) at re:Invent, these are the top services for developers to build, run, and grow their games.</p> 
<h2>Cloud Game Development</h2> 
<ul> 
 <li><strong>Amazon CodeCatalyst (Preview)</strong> – allows rapid launch of collaborative development environments based on blueprints. Blueprints create a tailored environment with project planning tools, repositories, coding IDE configuration, and build/test/deploy pipelines. This is a great first step in quickly starting a new project and easily bringing developers on board. <a href="https://www.youtube.com/watch?v=mKdGehrUYFI">Watch the re:Invent breakout session</a>.</li> 
</ul> 
<h2>Game Servers</h2> 
<ul> 
 <li><strong>Amazon GameLift Anywhere (New)</strong> – enables game developers to host GameLift fleet locations on their custom hardware, allowing a hybrid deployment model combining managed and self-hosted fleets. The feature also allows faster local iteration by registering development machine processes as game servers for a <a href="https://docs.aws.amazon.com/gamelift/latest/developerguide/fleets-creating-anywhere.html">GameLift Anywhere fleet</a>. Another great use case is for PC and console games that require compute flexibility (<a href="https://aws.amazon.com/eks/">Amazon Elastic Kubernetes Service (EKS)</a>, hybrid cloud paired with on-premises gear) with ability to burst into the cloud. <a href="https://www.youtube.com/watch?v=tCum8vhJE5g">Watch the demo</a>.</li> 
 <li><strong>Amazon ECS Task Scale-In Protection (New)</strong> – enables developers to protect long-running tasks from termination by scale-in events and deployments. With this feature, Amazon Elastic Container Service (Amazon ECS) Tasks running an active game session can protect themselves from termination by service auto scaling. This allows game developers to dynamically scale their ECS game server workloads while protecting active game sessions from termination on scale-in. <a href="https://aws.amazon.com/about-aws/whats-new/2022/11/amazon-ecs-task-scale-in-protection/">Learn more</a>.</li> 
 <li><strong>CloudWatch Cross-Account Observability (New)</strong> – helps developers monitor their game server workloads across games and game environments within their AWS organization. Previously, <a href="https://aws.amazon.com/cloudwatch/">Amazon CloudWatch</a> aggregated multi-region metrics, but this feature allows for aggregation and easy access for multiple games in different accounts. <a href="https://aws.amazon.com/blogs/aws/new-amazon-cloudwatch-cross-account-observability/">Learn more</a>.</li> 
 <li><strong>AWS Compute Optimizer (New)</strong> – the new partner metrics feature allows developers to combine data from AWS as well as commonly used partners such as Datadog. This data provides suggestions for right-sizing game server instances on EC2. <a href="https://aws.amazon.com/about-aws/whats-new/2022/11/aws-compute-optimizer-external-metrics-observability-partners/">Learn more</a>.</li> 
 <li><strong>AWS Local Zones (New Locations)</strong> – provides developers with low-latency game servers closer to their players. Now available in Buenos Aires, Copenhagen, Helsinki, and Muscat. <a href="https://aws.amazon.com/about-aws/whats-new/2022/11/aws-local-zones-buenos-aires-copenhagen-helsinki-muscat/">Learn more</a>.</li> 
</ul> 
<h2>Live Operations</h2> 
<ul> 
 <li><strong>Amazon GameSparks (Preview)</strong> – provides game developers with features for building, running, and scaling the backend for their games. Now available in preview in the AWS Asia Pacific (Tokyo) Region. <a href="https://aws.amazon.com/about-aws/whats-new/2022/11/amazon-gamesparks-available-aws-asia-pacific-tokyo-region-preview/">Learn more</a>.</li> 
 <li><strong>AWS Application Composer (Preview)</strong> – enables developers to drag, drop, and connect AWS services into an application architecture using browser-based visual canvas. This allows game developers to quickly experiment with serverless backends. <a href="https://www.youtube.com/watch?v=txzU1Go9oxc">Watch the breakout session</a>.</li> 
</ul> 
<h2>Game AI/ML</h2> 
<ul> 
 <li><strong>Amazon SageMaker Data Wrangler Supports SaaS Applications as Data Sources (New)</strong> – provides game developers the ability to pull in multiple data sources beyond <a href="https://aws.amazon.com/s3/">Amazon S3</a>, Snowflake, and Databricks to augment player telemetry data to allow more comprehensive player 360 analysis and machine learning (ML) modeling. <a href="https://aws.amazon.com/blogs/aws/new-amazon-sagemaker-data-wrangler-supports-saas-applications-as-data-sources/">Learn more</a>.</li> 
</ul> 
<h2>Bonus Service</h2> 
<ul> 
 <li><strong>AWS SimSpace Weaver (New)</strong> – highlighted in two keynotes, <a href="https://aws.amazon.com/simspaceweaver/">AWS SimSpace Weaver</a> is a new compute service to build, operate, and manage large-scale simulations on AWS. Initially designed for massive real-world scale simulations, it will provide features for future massively multiplayer experiences. <a href="https://www.youtube.com/watch?v=C3b-XFtiAn0">Watch the breakout session</a> and <a href="https://www.youtube.com/watch?v=lHA0LtS892I">demo</a> to learn more.</li> 
</ul> 
<h2>Customer Sessions</h2> 
<p>Game developers had a big presence at re:Invent, including spotlights in multiple keynotes and breakout sessions.</p> 
<ul> 
 <li><strong>Riot Games</strong> – presented two breakout sessions at re:Invent. <a href="https://www.youtube.com/watch?v=18Opqwna6gY">In this session</a>, the company shared its esports journey from a stopgap solution to laying the foundation for the next generation of esports fandom with <a href="https://www.youtube.com/watch?v=2YYEoatPKuc">Project Stryker</a>. <a href="https://www.youtube.com/watch?v=fJIkTH3HMAk&amp;t=54s">In its second session</a>, the company presented a solution that unifies cost, usage, recommendations, performance, governance, and security into a data lake and prescriptive reporting engine known as The Riot Act.</li> 
 <li><strong>Epic Games</strong> – highlighted in <a href="https://youtu.be/RfvL_423a-I">Dr. Werner Vogel’s keynote</a>, Epic Games announced its 2 million MetaHumans milestone and RealityScan app launch on AWS. MetaHuman Creator uses Amazon EC2 GPU instances to support the heavy lift in data processing needed to create real-time digital humans. RealityScan, a mobile application that takes images from the real world and turns them into 3D models ready to import into game engines, is an example of emerging artificial intelligence (AI) tools that will further democratize game development by allowing the creation of unique content without a massive art budget. <a href="https://aws.amazon.com/blogs/industries/epic-games-launches-realityscan-app-on-aws-at-reinvent/">Learn more</a>.</li> 
 <li><strong>Warner Bros. Games</strong> – highlighted in <a href="https://youtu.be/TL2HtX-FmiQ">Swami Sivasubramanian’s keynote</a>, Warner Bros. Games showcased how it leverages AWS to easily scale games like MultiVersus to handle up to three billion events per day at launch while using analytics to make data-driven decisions that influence business strategy and improve player experiences. <a href="https://aws.amazon.com/solutions/case-studies/warner-bros-games-video-testimonial/">Watch the video</a>.</li> 
 <li><strong>KakaoGames</strong> – led a breakout session on its learnings after launching the massive multiplayer online role-playing game (MMORPG) “Odin: Valhalla Rising” on AWS. It became one of the most successful games in Korean game industry history, with 400,000 daily active users and $2M in daily revenue. <a href="https://www.youtube.com/watch?v=0WekEiAmDFw">In this session</a>, KakaoGames explained how to maximize game server performance, how to architect MMORPGs on AWS, and why the company chose AWS.</li> 
 <li><strong>Creative Assembly (CA)</strong> – a SEGA studio, Creative Assembly has been at the forefront of the strategy gaming genre for 22 years with its award-winning Total War franchise. <a href="https://www.youtube.com/watch?v=ZT2a6mlEe34&amp;list=PL2yQDdvlhXf83Wd-5YinHHNdN73iLyWzV&amp;index=28">In this breakout session</a>, Creative Assembly explained how the cloud can help game studios deliver AAA games to large global audiences, how evolving game architecture can simplify operations, and how the cloud can help developers focus on delivering increasingly innovative game play.</li> 
</ul>

</ul>
