<p>We are excited to announce the general availability of GameLift Anywhere, a new <a href="https://aws.amazon.com/gamelift/">Amazon GameLift</a> feature that decouples game session management from underlying compute resources. With this release, Amazon GameLift can now deploy and manage dedicated game servers hosted on-premises, in the cloud, or through hybrid configurations. The flexibility to deploy and manage game servers hosted anywhere allows game developers to iterate faster during game development and to migrate to the cloud at their own pace.</p> 
<p>In 2016, Amazon Web Services (AWS) launched Amazon GameLift, a dedicated hosting solution that securely deploys and automatically scales fleets of session-based multiplayer game servers to meet worldwide player demand across&nbsp;<a href="https://aws.amazon.com/blogs/gametech/amazon-gamelift-is-now-easier-to-manage-fleets-across-regions/">multiple AWS Regions</a>&nbsp;and&nbsp;<a href="https://aws.amazon.com/blogs/gametech/amazon-gamelift-launches-support-for-aws-local-zones/">AWS Local Zones</a>. GameLift already includes standalone features to support low-cost game fleets with&nbsp;<a href="https://aws.amazon.com/blogs/aws/amazon-gamelift-fleetiq-and-spot-instances-save-up-to-90-on-game-server-hosting/">GameLift FleetIQ</a>&nbsp;and player matchmaking with&nbsp;<a href="https://aws.amazon.com/blogs/gametech/matchmaking-your-way-amazon-gamelift-flexmatch-and-game-session-queues/">GameLift FlexMatch</a>. Game developers asked us to reduce the wait time to deploy a candidate server build to the cloud to test and iterate their game during development. Customers also&nbsp;told us they often have ongoing bare-metal contracts or on-premises game servers and want the flexibility to use their existing infrastructure with cloud servers.</p> 
<p>With the new release, game developers can now integrate hardware anywhere into their GameLift game hosting environment. GameLift Anywhere introduces a compute type called “Anywhere” that is used to register on-premises or cloud hardware with GameLift in a GameLift Anywhere fleet. Anywhere fleets can be mixed-and-matched with fully managed Amazon GameLift EC2 fleets so developers don’t need to run and operate separate game server management software for on-premises and cloud workloads.</p>

<p>With GameLift Anywhere, developers can now:</p> 
<ul> 
 <li><strong>Develop games faster</strong> – Instantly test and iterate server builds in a local desktop environment while still leveraging GameLift FlexMatch and Queue services.</li> 
 <li><strong>Manage hybrid deployments</strong> – Deploy, operate, and scale dedicated game servers hosted in the cloud or on-premises, all from a single location.</li> 
 <li><strong>Streamline server operations</strong>&nbsp;– Reduce cost and operational complexity by unifying server infrastructure under a single game server orchestration layer.</li> 
</ul> 
<p><a href="https://www.nitrogames.com/">Nitro Games</a>, a Finnish games studio, has been an Amazon GameLift customer since 2020 and has used the service for player matchmaking and managing dedicated game servers in the cloud. Daniel Liljeqvist, Senior DevOps Engineer at Nitro Games said, “<em>With GameLift Anywhere, we can easily debug a game server on our local machine, saving us time and making the feedback loop much shorter when we are developing new games and features.</em>”</p>
