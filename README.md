# Amazon GameLift (amazon-gamelift)
Amazon GameLift is a dedicated game server hosting solution that deploys, operates, and scales cloud servers for multiplayer games. It provides low-latency, low-cost server infrastructure, eliminates operational overhead, and allows you to focus on creating great gaming experiences.

## APIs

### Amazon GameLift API
The Amazon GameLift API provides programmatic access to create and manage fleets, game sessions, player sessions, matchmaking configurations, and game server groups for hosting multiplayer game servers.
